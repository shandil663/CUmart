Cumart
Cumart is an Android app built for buying, selling, and renting items within the Chandigarh University campus.

Features
Buy: Find items you need from other students.
Sell: List your items to sell to the campus community.
Rent: Offer items for rent or find things to rent temporarily.

Getting Started
Clone this repository.
Open the project in Android Studio.
Connect your Firebase project.
Build and run!
